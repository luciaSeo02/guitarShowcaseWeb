# Handcrafted Guitars & Metal Art

A bilingual portfolio website developed for a local artisan business specializing in handcrafted guitars, restorations, and metal artwork.

The platform provides a premium digital presence focused on craftsmanship, minimal design, and custom order management.

---

## Overview

This project delivers:

- Service presentation
- Featured work showcase
- Custom order request workflow
- English / Spanish internationalization
- Responsive and performance-oriented UI

The architecture prioritizes scalability, maintainability, and clean component structure.

---

## Tech Stack

- React
- TypeScript
- React Router
- TailwindCSS
- react-i18next
- Vercel (deployment)

---

## Key Features

- Internationalization with persistent language selection
- Dynamic routing with state-based service selection
- Custom order form with API integration
- Modular component architecture
- Fully responsive layout

---

## Architecture Notes

- Structured translation system using JSON namespaces
- Component-driven UI organization
- Centralized language switching logic
- Clean separation between presentation and routing logic

---

## Deployment

Deployed using Vercel with automatic CI/CD from the main branch.
